## ğŸš€ Quick Start

### Prerequisites
- PHP 8.2 or higher
- Composer
- Node.js 18+ and npm
- SQLite database (included)
- Google OAuth credentials
- OpenAI API key

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd notes-ai
   ```

2. **Install PHP dependencies**
   ```bash
   composer install
   ```

3. **Install Node.js dependencies**
   ```bash
   npm install
   ```

4. **Environment setup**
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

5. **Configure environment variables** (see [Environment Configuration](#environment-configuration))

6. **Database setup**
   ```bash
   php artisan migrate
   php artisan db:seed  # Optional: seed with sample data
   ```

7. **Build frontend assets**
   ```bash
   npm run build
   ```

8. **Start the development server**
   ```bash
   php artisan serve --host=localhost --port=8000
   ```

   The application will be available at `http://localhost:8000`

## âš™ï¸ Environment Configuration

### Required Environment Variables

Copy `.env.example` to `.env` and configure the following:

#### Application Settings
```env
APP_NAME="AI Note Editor"
APP_ENV=local
APP_KEY=  # Generated by php artisan key:generate
APP_DEBUG=true
APP_URL=http://localhost:8000
```

#### Database Configuration
**SQLite Configuration:**
```env
DB_CONNECTION=sqlite
DB_DATABASE=database/database.sqlite
# No additional database credentials needed for SQLite
```

#### Google OAuth Setup
1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create a new project or select existing
3. Enable Google+ API
4. Create OAuth 2.0 credentials
5. Add authorized redirect URI: `http://localhost:8000/auth/google/callback`

```env
GOOGLE_CLIENT_ID=your-google-client-id
GOOGLE_CLIENT_SECRET=your-google-client-secret
GOOGLE_REDIRECT_URI=http://localhost:8000/auth/google/callback
```

#### OpenAI Configuration
1. Get API key from [OpenAI Platform](https://platform.openai.com/)
2. Configure in `.env`:

```env
OPENAI_API_KEY=your-openai-api-key
OPENAI_MODEL=gpt-4  # or your preferred model
```

## ğŸ“š API Documentation

### Authentication Endpoints
- `GET /auth/google` - Redirect to Google OAuth
- `GET /auth/google/callback` - Handle OAuth callback
- `POST /logout` - Logout user

### Notes API
All note endpoints require authentication.

- `GET /api/notes` - List all notes
- `GET /api/notes/{id}` - Get specific note
- `POST /api/notes` - Create new note
- `PATCH /api/notes/{id}` - Update note
- `DELETE /api/notes/{id}` - Delete note

### AI Enhancement
- `POST /api/ai/enhance` - Enhance text content
- `POST /api/ai/suggest` - Get content suggestions

## ğŸ†˜ Troubleshooting

### Common Issues

**Database Connection Issues**
- Ensure SQLite database file exists in `database/database.sqlite`
- Check file permissions for the database directory
- Verify `DB_CONNECTION=sqlite` in `.env`

**Google OAuth Issues**
- Verify redirect URI matches exactly
- Check Google Cloud Console configuration
- Ensure Google+ API is enabled

**Build Issues**
- Clear npm cache: `npm cache clean --force`
- Delete `node_modules` and reinstall
- Check Node.js version compatibility

**Performance Issues**
- Run `php artisan optimize`
- Enable caching in production
- Optimize database queries
